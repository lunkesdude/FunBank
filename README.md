# FunBank
🏦 FunBank — фан‑банк для игры в “финансы” прямо в браузере. Создаёшь аккаунт 👤, получаешь карту 💳, следишь за балансом 💰. Можно переводить деньги по нику или карте 💸, менять валюту 💱, покупать в маркете 🛒 и хранить покупки в инвентаре 🎒. Есть дейли‑бонус 🎁, вклад 🏦 и топ 🏆. PIN 🔒, аватар 🖼️ и темы 🎨!
